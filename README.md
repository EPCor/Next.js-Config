# Next.js Config

Next.js Engineering Configuration

## Usage

<a class="btn btn-sm btn-primary ml-2" href="https://github.com/EPCor/RN-Config/generate">Use this template</a> or clone this repository

```sh
$ npm i # Install
$ npm run dev # development
$ npm start # start
```

## dependencies

```json
{
  "next": "^9.5.3",
  "react": "^16.13.1",
  "react-dom": "^16.13.1"
}
```
